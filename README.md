## Open Source

**[graphify](https://github.com/safishamsi/graphify)** - AI coding assistant skill that turns any folder of code, docs, papers, images, or video into a queryable knowledge graph. Works with Claude Code, Codex, Gemini CLI, Cursor, Aider, Kiro, and 10+ more. 71.5x fewer tokens per query vs raw context.
